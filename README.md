# IFTENDO
Extention to the IFT-C3 board with OLED screen (128x64), I2C address : 0x3C, and buttons

SDA: GPIO5<br>
SCL: GPIO6

Keyboard: <br>
GPIO20 (RX) -> Output 1<br>
GPIO10 -> Output 2<br>
GPIO0  -> Output 3<br>
GPIO7  -> Input 1<br>
GPIO8  -> Input 2<br>
GPIO9  -> Input 3<br>

Up     : O_1, I_1<br>
Left   : O_1, I_2<br>
Down   : O_1, I_3<br>
Right  : O_2, I_1<br>
Select : O_2, I_2<br>
Start  : O_2, I_3<br>
A      : O_3, I_1<br>
B      : O_3, I_2
